#### 텍스트 파일 과 바이너리 파일
- 바이너리 파일 : 문자와 숫자가 저장되어 있는 파일
  - 데이터를 있는 그대로 읽고 쓴다
- 텍스트 파일 : 문자만 저장되어 있는 파일
  - 숫자를 문자로 변환 후 쓴다
  - 텍스트는 숫자 12를 문자 '1'과 '2'로 변경해서 처리
- 데이터 : 크게 숫자와 문자로 구성됨
- 메모장으로 열었을 때 알아 볼 수 있으면 텍스트 파일이고
알아 볼 수 없으면 바이너리 파일이다
